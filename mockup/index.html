claudex / remote-control 2026 · v0 product design

A remote control for the claude running on your own machine.

claudex runs a small service on any machine that has the Claude Code CLI installed, and exposes a polished browser interface — designed mobile-first — that mirrors the Desktop Code tab. Review diffs from the train, approve permissions from the kitchen, watch a long refactor from bed.
